# VPJoomla

**Lightweight, modern extensions for Joomla 5 & 6.**

Clean code · accessibility-first · built for performance.

[Website](https://vpjoomla.com) · [Documentation](https://vpjoomla.com/documentation) · [Support](mailto:support@vpjoomla.com)

---

We build free and premium Joomla extensions with a focus on performance, clean
code and great documentation. Every extension is tested on the latest Joomla 5
and Joomla 6, follows Joomla coding standards, and ships with a self-hosted
update server so your sites stay current.

## Why VPJoomla

- **Joomla 5 & 6 native** — no legacy baggage, modern APIs only.
- **Performance-first** — minimal footprint, no bloat.
- **Documented** — every extension comes with a full documentation set.
- **Maintained** — regular updates via our own update server.
